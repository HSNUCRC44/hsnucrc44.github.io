社團簡介

電子計算機研究社，也稱為電算社。社課以 C++、Python 為主。在師大附中是人數最多的學術性社團，有一間舊北樓*最大的社辦。課外活動有小社課、資訊營、茶會、秋遊、秋烤、寒訓等有趣的內容

*附中的社辦聚集地

*圖片：本社靜態展遊戲的主人公

充實活動

在附中電算不只有充實的課程，我們也有各式各樣的活動給社員參與。例如：資訊營、茶會、聯合迎新、秋遊、秋烤、耶晚、冬鍋、寒訓、以及跨社、跨校聯課等不定時出現的活動

社徽理念

在電路板的直線中尋找斜向的可能

大社課

大社課的時間是每個星期五第七節，是學校的社課時間。這時的課程主要以入門以及較簡單的內容為主。
我們會分三間電腦教室上不同的內容，大家可以自由去任何一間聽課。

C++ 語法班

介紹 C++ 基本語法與指令。並介紹 STL 的使用以及一些演算法的實踐

Python 語法班

Python 是近年流行的語言之一。本課程會帶大家熟悉 Python 的基礎，並且實作 Python 程式。 最後大家將可以用各種 Python 套件完成屬於自己的專案 (像是做遊戲)。

C++ 進階班

建議有一定 C++ 基礎的學員參加。

小社課

小社課是高二在其他時間 (平日晚上、假日) 舉行的社課，主題、內容不拘。

洋蔥的小社課

電算社 44th 學術長洋蔥開的小社課，內容包含資訊工程概論、CTF 競賽 (資安)、機器學習等內容，有無基礎的社員都歡迎參加，對於更深入的技術有興趣的社員更歡迎。詳細課表開學後公布

黎安的小社課 (暫定)

黎安是電算社 44th 的公關長。這個小社課將以很輕鬆的方式教學弟妹如何使用電資圈常用的筆記工具---HackMD以及如何交女朋友(????)等豐富的內容。

不定時出現的小社課

不定時會有人突然想要分享東西(?)我們也歡迎高一的社員上台開啟屬於你的小社課！
